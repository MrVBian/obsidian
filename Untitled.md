# Debug调试语句
```shell
# 查看当前topic
ros2 topic list

# 查看topic当前数据
ros2 topic echo /robot_pose_2d

# 查看topic类型
ros2 topic type /robot_pose_2d

# 管道过滤
ros2 topic list | grep robot
```

```shell
docker run -d \
  --name ha \
  --privileged \
  --restart=unless-stopped \
  -e TZ=Asia/Shanghai \
  -v /home/zme/Software/ha/config:/config \
  -v /run/dbus:/run/dbus:ro \
  --network=host \
  ghcr.io/home-assistant/home-assistant:stable
  
// recommend
docker run -d \
  --name ha \
  --privileged \
  --restart=unless-stopped \
  -e TZ=Asia/Shanghai \
  -e HTTP_PROXY="http://127.0.0.1:7897" \
  -e HTTPS_PROXY="http://127.0.0.1:7897" \
  -e NO_PROXY="localhost,127.0.0.1,192.168.0.0/16,10.0.0.0/8,*.local" \
  -v /home/zme/docker/ha/config:/config \
  -v /run/dbus:/run/dbus:ro \
  --network=host \
  ghcr.nju.edu.cn/home-assistant/home-assistant:stable
  
docker stop ha && docker rm ha
```
- `/PATH_TO_YOUR_CONFIG` 指向存储配置的文件夹，并运行容器。请确保保留 `:/config` 部分。
- D-Bus 是可选的，但如果您计划使用蓝牙集成，则必须启用。

**默认端口号**
- 前端地址：[http://127.0.0.1:8123](http://127.0.0.1:8123)

# 安装Docker-Clash
---
https://github.com/gangz1o/clash4docker
```shell
docker run -d \
  --name clash \
  --restart unless-stopped \
  -p 7890:7890 \
  -p 7891:7891 \
  -p 7897:7897 \
  -p 9090:9090 \
  -v /home/zme/docker/clash/config:/root/.config/mihomo \
  -e SUB_URL=https://8BRKzOs.doggygo.top:8443/api/v1/client/7f8ec7e3bb3c7cf4df4512fd2b02d17d \
  -e SUB_CRON="0 */6 * * *" \
  -e SECRET=Bian1024. \
  -e ALLOW_LAN=true \
  gangz1o/glash:latest
  
docker stop clash && docker rm clash
```
**首次访问需要配置**：
- 前端地址：[http://127.0.0.1:9090/ui/](http://127.0.0.1:9090/ui/)
- 后端地址：[http://127.0.0.1:9090](http://127.0.0.1:9090)
- 密钥：与 config.yaml 中的 `secret` 一致
```shell
alias po="export https_proxy=127.0.0.1:7897 && export http_proxy=127.0.0.1:7897 && export ftp_proxy=127.0.0.1:7897 && ALL_PROXY=127.0.0.1:7897"
alias pf="unset http_proxy https_proxy ftp_proxy ALL_PROXY"
```
**检测配置成功：**
```shell
curl -x http://127.0.0.1:7897 https://www.google.com -I
curl -x http://127.0.0.1:7897 https://raw.githubusercontent.com -I
```

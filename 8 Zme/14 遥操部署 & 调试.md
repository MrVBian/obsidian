# 迁移容器
## 在源机器上操作
### 1. 停止容器（可选但推荐）
```bash
docker stop xr
```
### 2. 将容器提交为新镜像
```bash
docker commit xr ros_humble_xr:v1
```
### 3. 将镜像保存为 tar 文件
```bash
docker save ros_humble_xr:v1 -o ros_humble_xr.tar
```
### 4. 传输文件到目标机器
```bash
# 使用 scp 传输（示例）
scp ros_humble_xr.tar zme@ip:/path/to/destination/
```
## 在目标机器上操作
### 5. 加载镜像
```bash
docker load -i ros_humble_xr.tar
```
### 6. 运行新容器
```bash
docker run -d \
  --name xr \
  ros_humble_xr:latest \
  /ros_entrypoint.sh bash
```

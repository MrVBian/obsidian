# 1. 流程图结构

定义一个完整的流程图（顶级结构）。

```json
{
    "task_info": {},
    "nodes": [],
}
```

| 字段        | 类型   | 描述           |
| ----------- | ------ | -------------- |
| `task_info` | object | 流程图信息     |
| `nodes`     | array  | 流程图节点数组 |

# 2 流程图信息

```json
{
    "task_info": {
    	"name": "course_1",
    	"version": "1.0",
	}
}
```

| 字段      | 类型   | 描述                    |
| --------- | ------ | ----------------------- |
| `name`    | string | 流程名称。例如课程名。  |
| `version` | string | 版本号（大版本_小版本） |

# 3 流程图节点

## 3.1 节点通用属性

```json
{
    "nodes": [
        {
            "type": "",
            "params": {},
        },
        {
            "type": "",
            "params": {},
        },
    ],
}
```

所有节点均包含以下基础字段。

| 字段     | 类型   | 描述                                                         |
| -------- | ------ | ------------------------------------------------------------ |
| `nodes`  | array  | 节点对象数组                                                 |
| `type`   | string | `节点类型`（与`节点特定参数`对应，见**[3.2 节点类型规范](#32-节点类型规范)**） |
| `params` | object | `节点特定参数`（与`节点类型`对应，见**[3.2 节点类型规范](#32-节点类型规范)**） |

## 3.2 节点类型规范

### Ⅰ机器人规控

**节点说明：机器人移动方向**

**节点类型：`robot_move_direction`**

```json
{
    "type": "robot_move_direction",
    "params": {
        "direction": "forward"
    }
}
```

| 字段        | 类型   | 描述                                                         |
| ----------- | ------ | ------------------------------------------------------------ |
| `direction` | string | 移动方向：`forward`（前）、`backward`（后）、`left`（左）、`right`（右） |

**节点说明：机器人移动**

**节点类型：`robot_move`**

```json
{
    "type": "robot_move",
    "params": {
    	"distance": 3.0
    },
}
```

| 字段       | 类型   | 描述             |
| ---------- | ------ | ---------------- |
| `distance` | double | 移动距离，单位米 |

**节点说明：机器人旋转**

**节点类型：`robot_rotate`**

```json
{
    "type": "robot_rotate",
    "params": {
        "degrees": 15.0
    }
}
```

| 字段      | 类型   | 描述                                           |
| --------- | ------ | ---------------------------------------------- |
| `degrees` | double | 机器人旋转角度，顺时针右转为正方向，单位：角度 |

**节点说明：机器人升降**

**节点类型：`robot_lift`**

```json
{
    "id": "",
    "type": "robot_lift",
    "params": {
        "height": 0.0
    }
}
```

| 字段     | 类型   | 描述                           |
| -------- | ------ | ------------------------------ |
| `height` | double | 升降高度，升高为正方向，单位米 |

**节点说明：机器人头部旋转**

**节点类型：`robot_gimbal_rotate`**

```json
{
    "type": "robot_head_rotate",
    "params": {
        "gimbal1": 0.0,
        "gimbal2": 0.0
    }
}
```

| 字段      | 类型   | 描述                                          |
| --------- | ------ | --------------------------------------------- |
| `gimbal1` | double | 云台1旋转角度，逆时针左转为正方向，单位：角度 |
| `gimbal1` | double | 云台2旋转角度，抬头为正方向，单位：角度       |

### Ⅱ 音频控制

**节点说明：机器人说出文字**

**节点类型：`robot_speak_text`**

```json
{
    "type": "robot_speak_text",
    "params": {
        "message": "你好！"
    }
}
```

| 字段      | 类型   | 描述       |
| --------- | ------ | ---------- |
| `message` | string | 说话的内容 |

**节点说明：机器人音量设置**

**节点类型：`robot_volume_level`**

```json
{
    "type": "robot_volume_level",
    "params": {
        "level": 100
    }
}
```

| 字段    | 类型 | 描述                  |
| ------- | ---- | --------------------- |
| `level` | int  | 音量值，范围 [0, 100] |

**节点说明：机器人音量变化**

**节点类型：`robot_volume_change`**

```json
{
    "type": "robot_volume_change",
    "params": {
        "volume": 10
    }
}
```

| 字段     | 类型 | 描述                     |
| -------- | ---- | ------------------------ |
| `volume` | int  | 音量变化的值（可正可负） |

**节点说明：播放指定音频（等待播完）**

**节点类型：`robot_play_audio_sync`**

```json
{
    "type": "robot_play_audio_sync",
    "params": {
        "sound": "music1"
    }
}
```

| 字段    | 类型   | 描述                            |
| ------- | ------ | ------------------------------- |
| `audio` | string | 音频名称，如 `music1`、`music2` |

**节点说明：播放指定声音（不等待播完）**

**节点类型：`robot_play_audio_async`**

```json
{
    "type": "robot_play_audio_async",
    "params": {
        "sound": "music1"
    }
}
```

| 字段    | 类型   | 描述                            |
| ------- | ------ | ------------------------------- |
| `audio` | string | 音频名称，如 `music1`、`music2` |

**节点说明：关闭所有音频**

**节点类型：`robot_stop_audio`**

```json
{
    "type": "robot_stop_audio",
    "params": {
    }
}
```

### Ⅲ 逻辑语句

**节点说明：循环语句**

**节点类型：`while`**

```json
{
    "type": "while",
    "params": {
        "loop_count": 10,
        "nodes": [
            {
                "type": "",
                "params": {},
            },
            {
                "type": "",
                "params": {},
            },
        ],
    }
}
```

| 字段         | 类型  | 描述                   |
| ------------ | ----- | ---------------------- |
| `loop_count` | int   | 循环次数。-1为无限循环 |
| `nodes`      | array | 节点对象数组           |

# 迁移容器
## 在源机器上操作
### 1. 停止容器（可选但推荐）
```bash
docker stop xr
```
### 2. 将容器提交为新镜像
```bash
docker commit xr ros_humble_xr_agx:v1

docker commit xr ros_humble_xr_zme:v2
```
### 3. 将镜像保存为 tar 文件
```bash
docker save ros_humble_xr_agx:v1 -o ros_humble_xr_agx.tar

docker save ros_humble_xr_zme:v2 -o ros_humble_xr_zme.tar
```
### 4. 传输文件到目标机器
```bash
# 使用 scp 传输（示例）
scp ros_humble_xr_zme.tar zme@ip:/path/to/destination/
```
## 在目标机器上操作
### 5. 加载镜像
```bash
docker load -i ros_humble_xr_agx.tar

docker load -i ros_humble_xr_zme.tar
```


### 6. 运行新容器
```bash
docker run -it -d --name xr \
  -w /projects \
  -v /home/zme/pico_tele_agx:/projects \
  -v /dev/bus/usb:/dev/bus/usb \
  --privileged \
  --net=host \
  ros_humble_xr_agx:v1

docker run -it -d --name xr \
  -w /projects \
  -v /home/zme/pico_tele_zme:/projects \
  -v /home/zme/robot_config/system_config/zme_robot/cyclonedds_config:/projects/cyclonedds_config \
  -v /dev/bus/usb:/dev/bus/usb \
  --privileged \
  --net=host \
  ros_humble_xr_zme:v2
  
  
source /opt/ros/humble/setup.bash
export RMW_IMPLEMENTATION=rmw_cyclonedds_cpp
export CYCLONEDDS_URI=file:///projects/cyclonedds_config/cyclonedds.xml
export ROS_DOMAIN_ID=29
```

### 7 安装docker
```shell
sudo apt update
sudo apt install -y docker.io

sudo groupadd docker               #添加用户组
sudo usermod -aG docker ${USER}    #将当前用户添加至用户组
newgrp docker                      #更新用户组
sudo service docker restart
```
